## 🚀 Como Executar a Etapa 3

Siga os passos abaixo para rodar a análise de anomalias localmente ou via nuvem:

### 1. Pré-requisitos e Arquivos Necessários
Antes de iniciar, certifique-se de ter baixado os seguintes arquivos deste repositório:
* `etapa3_completa(2).ipynb` (Notebook do projeto)
* `comprovantes_pix_10000_anomalias.csv` (Base de dados)

### 2. Ambiente de Execução
Recomendamos a utilização do **Google Colab** para a execução deste projeto, evitando assim qualquer problema com a instalação local de dependências e bibliotecas.

### 3. Passo a Passo para Execução

1. **Abrir o Notebook:** Acesse o [Google Colab](https://colab.research.google.com/) e faça o upload do arquivo `etapa3_completa(2).ipynb`.
2. **Upload dos Dados:** * No painel lateral esquerdo do Colab, clique no ícone de **Pasta** (Arquivos).
   * Arraste ou faça o upload do arquivo `comprovantes_pix_10000_anomalias.csv` para esse diretório.
3. **Executar o Código:** Com o arquivo de dados carregado, basta executar as células do notebook na ordem exata em que foram escritas (de cima para baixo).
