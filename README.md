# Etapa3_Fin_Pessoais_Pix

Para executar essa etapa basta baixar o arquivo etapa3_completa(2).ipynb,fazer o download dos dados: comprovantes_pix_10000_anomalias.csv.
Ambiente de Execução: Recomendamos a utilização do Google Colab para evitar problemas de dependências.
Upload dos Dados: Abra o arquivo .ipynb no Colab , antes de executar as células, faça o upload do arquivo .csv no painel de arquivos lateral do Colab (ícone de pasta).
Em seguida executar os comando das células na ordem em que foram escritos.
Não tem a necessidade de mais nenhuma interação além disso.
